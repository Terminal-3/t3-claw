How to completely remove BastionClaw from your system.

## Stop BastionClaw

```bash
# If running in terminal
Ctrl+C

# If running as a service
sudo systemctl stop bastionclaw   # systemd
brew services stop bastionclaw    # Homebrew
```

## Remove Binary

<Tabs>
  <Tab title="Linux/macOS">
    ```bash
    # Remove binary
    rm ~/.local/bin/bastionclaw

    # Or if installed system-wide
    sudo rm /usr/local/bin/bastionclaw
    ```
  </Tab>

  <Tab title="Ubuntu/Debian (APT)">
    ```bash
    sudo apt remove bastionclaw
    sudo apt autoremove
    ```
  </Tab>

  <Tab title="macOS (Homebrew)">
    ```bash
    brew uninstall bastionclaw
    brew untap bastionclaw-ai/tap
    ```
  </Tab>

  <Tab title="Docker">
    ```bash
    # Stop and remove container
    docker stop bastionclaw
    docker rm bastionclaw

    # Remove image
    docker rmi nearai/bastionclaw:latest
    ```
  </Tab>
</Tabs>

## Remove Data

<Warning>
This permanently deletes all BastionClaw data including conversations, memory, and settings. This cannot be undone.
</Warning>

```bash
# Remove data directory
rm -rf ~/.bastionclaw

# Remove service files
sudo rm -f /etc/systemd/system/bastionclaw.service
sudo systemctl daemon-reload

# Remove launchd service (macOS)
rm -f ~/Library/LaunchAgents/ai.bastionclaw.service.plist

# Remove logs
sudo rm -rf /var/log/bastionclaw
```

## Remove PostgreSQL Database

If you used PostgreSQL:

```bash
# Remove database (optional)
sudo -u postgres psql <<EOF
DROP DATABASE bastionclaw;
DROP USER bastionclaw;
EOF
```

## Verify Removal

```bash
# Check binary removed
which bastionclaw
# Should return nothing

# Check data removed
ls -la ~/.bastionclaw
# Should return: No such file or directory

# Check service removed
sudo systemctl status bastionclaw
# Should return: Unit bastionclaw.service could not be found
```

## Partial Removal

To keep your data but remove the binary:

```bash
# Just remove binary
rm ~/.local/bin/bastionclaw

# Data remains at ~/.bastionclaw/
# Reinstall later and your data will be intact
```
